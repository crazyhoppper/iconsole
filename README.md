##iconsole: better console for nodejs

This item is used for print beautifully for console

You can use it to change all the methor like console.log or console.error

### install
	npm install iconsole

### usage
```
	require('iconsole')
	console.config({
		'prefix': 'iconsole-'
	})
	console.log('test')
```
